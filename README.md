# WSearch

WSearch 是一个基于 Tauri + React + TypeScript 的桌面搜索应用。

## 项目初始化

```bash
# 安装依赖
npm install -g pnpm

pnpm create tauri-app

pnpm add antd

pnpm install

# 安装 ant-design skill
pnpx skills add ant-design/antd-skill

# 启动开发服务器
pnpm tauri dev
```
